CSS
    .gallery {
        display: flex;
        flex-direction: row;
        flex-wrap: wrap;
        justify-content: center;
        align-items: center;
        gap: 16px;
        max-width: 1400px;
        margin: 0 auto;
        padding: 20px 10px;
    }
        display: flex; 
            → Mengaktifkan Flexbox.
        flex-direction: row; 
            → Gambar berbaris horizontal (______)
        flex-wrap: wrap; 
            → Jika layar kecil, gambar turun ke bawah.
        justify-content: center; 
            → Memposisikan gambar ke tengah.
        gap: 16px; 
            → Memberi jarak antar gambar.
        max-width: 1400px; 
            → Maksimal lebar galeri 1400px.
        margin: 0 auto; 
            → Galeri berada di tengah halaman.

    .gallery img {
        width: 100%;
        max-width: 350px;
        height: 300px;
        object-fit: cover;
        border-radius: 10px;
    }
        Lebar gambar penuh (width: 100%) dengan batas maksimum 350px.
        Tinggi gambar tetap 300px (height: 300px;).
        object-fit: cover; → Agar gambar memenuhi sebuah ukuran yang telah ditetapkan
        border-radius: 10px; → Membuat sudut gambar sedikit membulat.

    .gallery::after {
        content: "";
        width: 350px;
    }

    Dalam konteks ini, ::after menciptakan elemen kosong dengan lebar 350px 
    setelah semua gambar di dalam .gallery
