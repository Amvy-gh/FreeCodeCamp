CSS

.divider{
    border-bottom: 1px solid #080808;
    margin: 2px 0;
}

    border-bottom untuk membuat tebal garisnya 
    margin untuk space antara atas dan bawahnya 

.daily-value p:not(.no-divider) {
    border-bottom: 1px solid #888989;
}

    .daily-value p → Semua elemen <p> di dalam elemen dengan kelas .daily-value akan dipilih.
        :not(.no-divider) → Mengecualikan <p> yang memiliki kelas .no-divider.

<span></span> 
    -> digunakan ketika ingin membaut editan diantara element padding

text-indent
    -> berarti jika
        - Nilai positif (text-indent: 8px;) akan membuat baris pertama teks menjorok ke dalam sejauh 8 piksel.
        - Nilai negatif (text-indent: -8px;) akan membuat baris pertama teks keluar sejauh 8 piksel ke kiri
