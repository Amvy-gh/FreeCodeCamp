# CSS Styling Notes

## Link States Styling
```css
a:visited {
    color: black; /* Warna setelah link dikunjungi */
}
  
a:hover {
    color: brown; /* Warna saat link dihover */
}
  
a:active {
    color: brown; /* Warna saat link ditekan */
}
```

## Price Styling
```css
.price {
    width: 25%;   /* Atur lebar 25% agar tidak melebar */
    text-align: right;  /* Teks rata kanan */
}
